# Scientific Calculator

A beautiful, fully-functional scientific calculator built with Next.js 15 and Tailwind CSS.

## Features

### Basic Operations
- Addition, Subtraction, Multiplication, Division
- Decimal point support
- Sign change (+/-)
- Backspace
- Clear

### Scientific Functions
- **Trigonometry**: sin, cos, tan (with RAD/DEG toggle)
- **Logarithms**: ln (natural log), log (base 10)
- **Powers**: x², x³, x^y, e^x, 10^x
- **Roots**: √x
- **Special**: 1/x, |x|, n! (factorial), mod

### Memory Functions
- MC (Memory Clear)
- MR (Memory Recall)
- M+ (Memory Add)
- M- (Memory Subtract)

### Constants
- π (Pi)
- e (Euler's number)

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build for Production

```bash
npm run build
npm start
```

## Tech Stack

- **Next.js 15** - React framework with App Router
- **TypeScript** - Type safety
- **Tailwind CSS** - Utility-first styling
- **React 18** - UI library

## Design Features

- Beautiful gradient background
- Smooth animations and transitions
- Responsive design
- Modern glassmorphic UI elements
- Color-coded button groups for easy navigation
- Real-time calculation display
- Memory indicator

## Usage

1. **Basic Calculations**: Click number buttons and operations
2. **Scientific Functions**: Use the colored function buttons
3. **Toggle Angle Mode**: Switch between Radians (RAD) and Degrees (DEG)
4. **Memory**: Store and recall values using memory functions
5. **Constants**: Insert π or e with a single click
